# Mentions de tiers

Le décodage du format TArc1.10 et la liste `resources/tanuki.lst` s'appuient sur les travaux du projet GARbro de morkt, distribué sous licence MIT. Le texte de cette licence est fourni dans `licenses/GARbro_LICENSE.txt`.

Le chiffrement Blowfish est fourni à l'exécution par PyCryptodome. Pillow est utilisé pour vérifier les dimensions des images. Leurs licences respectives sont incluses par PyInstaller dans la distribution générée et sont disponibles dans leurs projets officiels.
